# Data Classification Policy

## 1. Classification Levels

| Level | Description | Examples |
|-------|-------------|----------|
| Public | No restrictions | Marketing materials |
| Internal | Business use only | Internal memos |
| Confidential | Restricted access | Financial data, PII |
| Restricted | Highest protection | Trade secrets, PHI |

## 2. Handling Requirements
- Public: No special handling
- Internal: Don't share externally
- Confidential: Encrypt in transit/rest, access control
- Restricted: Encrypt always, MFA, audit logging, DLP

## 3. Labeling
All documents must be labeled with classification level.
