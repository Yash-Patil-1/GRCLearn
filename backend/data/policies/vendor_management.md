# Vendor Management Policy

## 1. Purpose
Manage security risks from third-party vendors and service providers.

## 2. Vendor Assessment
- Security questionnaire for all new vendors
- Annual reassessment for critical vendors
- SOC 2 or ISO 27001 certification preferred

## 3. Contracts
- Include data protection clauses
- Right-to-audit provisions
- Breach notification requirements (72 hours)
- Data return/destruction on termination

## 4. Monitoring
- Review vendor security posture quarterly
- Monitor for vendor breaches
- Maintain vendor risk register
